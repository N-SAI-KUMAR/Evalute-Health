# Evalute Health

## Smart Hospital Appointment Management System

Evalute Health is a full-stack web application developed to simplify hospital appointment management. The system provides role-based access for administrators, doctors, and patients, enabling efficient management of appointments, prescriptions, and healthcare records.

## Features

### Authentication & Authorization

* Secure Login and Registration
* Role-Based Access Control
* JWT Authentication

### Patient Management

* Add, View, Update, and Delete Patients
* Patient Information Management

### Doctor Management

* Add, View, Update, and Delete Doctors
* Doctor Specialization Tracking

### Appointment Management

* Schedule Appointments
* View Appointment History
* Appointment Status Management

### Prescription Management

* Create Prescriptions
* View Prescription Records
* Manage Patient Treatment Information

### API Documentation

* Swagger API Documentation Integration

## Technology Stack

## Installation

### Clone Repository

```bash
git clone https://github.com/N-SAI-KUMAR/Evalute-Health.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
